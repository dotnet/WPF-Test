REQUIREMENTS
------------
To build the reporting systems, the following dependencies need to be on the
machine.

Whidbey
Latest version. See http://avalon/build/.

Rascal
Version matching the Whidbey version.

Product Studio 2.x
Install from http://productstudio/.

Magellan Framework
Install from http://magellan/. The code coverage tools are sufficient.
You might also be able to install from \\tamales\.

BUILDING
--------
To build from the command line rather than from VS or Rascal, the above is
required, and the following command should be used from the 
WinFundamentalReports directory.

c:\windows\Microsoft.NET\Framework\v2.0.50215\msbuild

The version change may change in the future.
